# 📊 주식계산기 (stock-calculator)

주식 투자에 필요한 핵심 계산기를 한 곳에서 제공하는 웹 서비스입니다.  
수익률, 평단가, 손절가, 목표가 등 다양한 계산을 빠르게 확인할 수 있습니다.

👉 사이트 바로가기  
https://주식계산기.kr

---

## 🚀 주요 기능

- 📈 주식 수익률 계산기
- 📊 평단가 계산기 (물타기 계산)
- 🛑 손절가 계산기
- 🎯 목표가 계산기
- 💰 배당 및 기타 투자 계산기

---

## 🧭 사이트 구조

- `/` → 메인 페이지
- `/stocks` → 주식 계산기 모음 (허브)
- `/profit-calculator` → 수익률 계산기
- `/average-price-calculator` → 평단가 계산기
- `/stop-loss-calculator` → 손절가 계산기
- `/target-price-calculator` → 목표가 계산기
